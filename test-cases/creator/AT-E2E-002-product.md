# AT-E2E-002: Create Digital Product

**Feature:** @products
**Role:** @creator
**Priority:** @smoke

## Steps

1. Goto products page
2. Click "Add Product" button
3. Fill product name
4. Fill product description
5. Set price to 29.99
6. Select category "Digital"
7. Click "Save" button
8. Wait for success toast

## Expected

- Toast shows "Product created successfully"
- New product appears in products list

## Test Data

- name: "E-Book"
- description: "A comprehensive digital guide"
- price: 29.99
- category: "digital"
